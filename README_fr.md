Jot est un éditeur Markdown multiplateforme pour macOS et iOS, conçu pour rester réactif lors de l'écriture en streaming — que ce soit en cours de frappe, collage ou rédaction par une IA token par token.

## Fonctionnalités

- **Aperçu Markdown en temps réel** — Modes édition, aperçu et partagé
- **Optimisé pour le streaming** — Reste fluide pendant l'écriture token par token par l'IA
- **Coloration syntaxique** — Basée sur Tree-sitter pour Markdown et blocs de code
- **Gestion native des fichiers** — DocumentGroup + FileDocument pour les opérations macOS standard
- **Barre latérale** — Arborescence et panneau de plan
- **Préférences** — Taille de police, famille de police, hauteur de ligne, export/import des préférences

## Configuration requise

- macOS 14.0+
- Xcode 15+
