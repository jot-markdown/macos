Jot è un editor Markdown multipiattaforma per macOS e iOS, progettato per rimanere reattivo durante la scrittura in streaming — che sia durante la digitazione, l'incollaggio o la scrittura token per token da parte di un'IA.

## Funzionalità

- **Anteprima Markdown in tempo reale** — Modalità modifica, anteprima e divisa
- **Ottimizzato per lo streaming** — Rimane fluido durante la scrittura token per token dell'IA
- **Evidenziazione della sintassi** — Basato su Tree-sitter per Markdown e blocchi di codice
- **Gestione nativa dei file** — DocumentGroup + FileDocument per operazioni file macOS standard
- **Barra laterale** — Albero dei file e pannello struttura
- **Preferenze** — Dimensione font, famiglia font, altezza riga, esporta/importa preferenze

## Requisiti

- macOS 14.0+
- Xcode 15+
