Jot é um editor Markdown multiplataforma para macOS e iOS, projetado para manter a responsividade durante a escrita em streaming — seja digitando, colando ou escrevendo token por token por uma IA.

## Funcionalidades

- **Visualização Markdown em tempo real** — Modos de edição, visualização e dividido
- **Otimizado para streaming** — Permanece fluido durante a escrita token por token da IA
- **Destaque de sintaxe** — Baseado em Tree-sitter para Markdown e blocos de código
- **Gerenciamento nativo de arquivos** — DocumentGroup + FileDocument para operações padrão macOS
- **Barra lateral** — Árvore de arquivos e painel de estrutura
- **Preferências** — Tamanho da fonte, família da fonte, altura da linha, exportar/importar preferências

## Requisitos

- macOS 14.0+
- Xcode 15+
