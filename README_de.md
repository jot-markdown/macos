Jot ist ein plattformübergreifender Markdown-Editor für macOS und iOS, der für Streaming-Szenarien optimiert ist — bleibt responsiv, wenn Text Token für Token von einer KI geschrieben wird.

## Funktionen

- **Echtzeit Markdown-Vorschau** — Bearbeiten-, Vorschau- und Splitscreen-Modus
- **Streaming-optimiert** — Bleibt während KI-Token-Schreibvorgängen flüssig
- **Syntax-Hervorhebung** — Tree-sitter-basiert für Markdown und Code-Blöcke
- **Native Dateiverarbeitung** — DocumentGroup + FileDocument für standard macOS-Dateioperationen
- **Seitenleiste** — Dateibaum und Gliederungsansicht
- **Einstellungen** — Schriftgröße, Schriftfamilie, Zeilenhöhe, Einstellungen exportieren/importieren

## Anforderungen

- macOS 14.0+
- Xcode 15+
