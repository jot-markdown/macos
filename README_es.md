Jot es un editor Markdown multiplataforma para macOS e iOS, optimizado para flujos de escritura en streaming — permanece responsive incluso cuando texto es escrito token a token por una IA.

## Características

- **Vista previa Markdown en tiempo real** — Modos de edición, vista previa y división
- **Optimizado para streaming** — Responde sin problemas durante escritura token por token de IA
- **Resaltado de sintaxis** — Basado en Tree-sitter para Markdown y bloques de código
- **Gestión nativa de archivos** — DocumentGroup + FileDocument para operaciones estándar de macOS
- **Barra lateral** — Árbol de archivos y panel de esquema
- **Preferencias** — Tamaño de fuente, familia de fuente, altura de línea, exportación/importación de preferencias

## Requisitos

- macOS 14.0+
- Xcode 15+
