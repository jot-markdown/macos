Jot은 macOS 및 iOS를 위한 스트리밍 인식 Markdown 편집기입니다. AI가 토큰을 하나씩 작성할 때도 항상 반응성을 유지하도록 설계되었습니다.

## 기능

- **실시간 Markdown 미리보기** — 편집, 미리보기, 분할 보기 모드
- **스트리밍 인식** — AI 토큰 작성 중에도 부드럽게 반응
- **구문 하이라이팅** — Tree-sitter 기반 Markdown 및 코드 블록 지원
- **네이티브 파일 처리** — DocumentGroup + FileDocument로 표준 macOS 파일 작업
- **사이드바** — 파일 트리 및 개요 패널
- **설정** — 글꼴 크기, 글꼴, 줄 높이, 설정 내보내기/가져오기

## 요구 사항

- macOS 14.0+
- Xcode 15+
