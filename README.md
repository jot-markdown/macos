Jot is a focused, streaming-aware Markdown editor for macOS and iOS — designed to stay responsive when text is being typed, pasted, or written by an AI a token at a time.

## Features

- **Real-time Markdown preview** — Edit, preview, or split view modes
- **Streaming-aware** — Stays responsive during AI token-by-token writing
- **Syntax highlighting** — Tree-sitter powered for Markdown and code blocks
- **Native file handling** — DocumentGroup + FileDocument for standard macOS file operations
- **Sidebar** — File tree and outline panel
- **Settings** — Font size, font family, line height, and export/import preferences

## Requirements

- macOS 14.0+
- Xcode 15+
