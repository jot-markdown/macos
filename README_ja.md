Jot は、macOS と iOS 向けのストリーミング対応 Markdown エディタです。AI がトークンを 1 つずつ書き込む際でも、常に応答性を維持します。

## 機能

- **リアルタイム Markdown プレビュー** — 編集、プレビュー、分割ビューの 3 モード
- **ストリーミング対応** — AI のトークン書き込み中もスムーズ
- **構文ハイライト** — Tree-sitter による Markdown とコードブロック対応
- **ネイティブファイル処理** — DocumentGroup + FileDocument で標準 macOS ファイル操作
- **サイドバー** — ファイルツリーとアウトラインパネル
- **設定** — フォントサイズ、フォント、行の高さ、設定のエクスポート/インポート

## 必要環境

- macOS 14.0+
- Xcode 15+
