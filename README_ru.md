Jot — это кроссплатформенный Markdown-редактор для macOS и iOS, оптимизированный для потоковой записи — остаётся отзывчивым даже когда текст вводится токен за токеном ИИ.

## Функции

- **Предпросмотр Markdown в реальном времени** — Режимы редактирования, просмотра и разделённого вида
- **Оптимизация для потоковой записи** — Остаётся плавным при побуквенном вводе ИИ
- **Подсветка синтаксиса** — На основе Tree-sitter для Markdown и блоков кода
- **Нативная работа с файлами** — DocumentGroup + FileDocument для стандартных операций macOS
- **Боковая панель** — Дерево файлов и панель структуры
- **Настройки** — Размер шрифта, семейство шрифтов, высота строки, экспорт/импорт настроек

## Требования

- macOS 14.0+
- Xcode 15+
