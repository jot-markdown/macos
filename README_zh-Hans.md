Jot 是一款专注于流式写入的跨平台 Markdown 编辑器，支持 macOS 和 iOS — 即使在 AI 逐 token 写入文本时也能保持流畅响应。

## 功能特性

- **实时 Markdown 预览** — 支持编辑、预览、分屏三种视图模式
- **流式写入优化** — AI 逐 token 写入时依然保持流畅响应
- **语法高亮** — 基于 Tree-sitter 的 Markdown 和代码块高亮
- **原生文件处理** — 使用 DocumentGroup + FileDocument 实现标准 macOS 文件操作
- **侧边栏** — 文件树和大纲面板
- **偏好设置** — 字体大小、字体族、行高，以及设置的导入/导出

## 系统要求

- macOS 14.0+
- Xcode 15+
